# CoinPusher

# Technology

- React
- Express
- Pusher

## Setup

Install the dependencies

```sh
$ npm install
```

Start app

```sh
$ npm start
```

### If you want to test app offline

Install serve globally

```sh
$ npm i -g serve
```

Build app

```sh
$ npm run build
```

Serve app

```sh
$ serve -s build
```

Point your browser to http://localhost:5000/
